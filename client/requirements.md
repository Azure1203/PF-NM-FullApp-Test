framer-motion | Smooth animations for drag-and-drop and page transitions

tailwind-merge | Utility for merging Tailwind classes safely

## Notes
The application requires a cohesive "dashboard" feel.
Uploads are handled via multipart/form-data to /api/orders/upload.
Syncing to Asana is a backend process triggered via POST /api/orders/:id/sync.
Fonts should be modern and clean - likely Inter or Plus Jakarta Sans.
