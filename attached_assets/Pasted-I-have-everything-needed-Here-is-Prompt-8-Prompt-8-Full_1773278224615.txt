Background: What the Asana integration does

Manual sync (Sync to Asana button):

    Creates a new Asana task in the PERFECT FIT PRODUCTION project, or updates an existing one if the project was auto-imported
    Sets the task name to the project name
    Populates all custom fields (see full list below)
    Writes task notes with the app URL and per-file job numbers
    Saves the Asana task GID to the local project record

Status sync (pull from Asana):

    Reads PF ORDER STATUS, PF PRODUCTION STATUS (multi-select), CIENAPPS JOB NUMBER, and the task's current section back into the local project record
    Called on demand and by the background sync-all job

Auto-import scheduler:

    Polls the READY TO IMPORT section of the NEW JOBS Asana project every 10 minutes
    For each unprocessed task with CSV attachments, downloads the CSVs and creates a project exactly like a manual upload
    Runs the full pipeline: countPartsFromCSV, createOrderFile, pricing loop, extractCTSParts, generateHardwareChecklistForFile, generatePackingSlipChecklistForFile, computeAutoProductionStatuses
    After creating the project, updates the Asana task with notes and custom fields
    Records the task GID in processed_asana_tasks to avoid re-importing
