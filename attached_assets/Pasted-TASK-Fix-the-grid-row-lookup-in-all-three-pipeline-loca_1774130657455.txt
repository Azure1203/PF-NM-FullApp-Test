Step 1 — Confirm or add findGridRowInCache in each file.

  function findGridRowInCache(
    cache: Map<number, any[]>,
    gridId: number,
    lookupValue: string,
    lookupColumn: string
  ): any | null

Lookup logic:
  1. rows = cache.get(gridId) — if undefined, return null
  2. Exact match: rows.find(r => r.lookupKey === lookupValue)
  3. Case-insensitive: rows.find(r => r.lookupKey?.toLowerCase() === lookupValue.toLowerCase())
  4. rowData column fallback: rows.find(r => {
       const rd = r.rowData as Record<string, any>;
       return rd && Object.values(rd).some(v => String(v).toLowerCase() === lookupValue.toLowerCase());
     })
  5. Return match or null

  const allGrids = await storage.getAttributeGrids();

    gridRowCache.set(grid.id, rows);
  }));
